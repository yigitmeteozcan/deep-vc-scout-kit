## Rules for GPT

When scouting:
1. Do not use the sources listed above as primary discovery channels.
2. If a company appears only in these sources with no other evidence, treat it as a signal that it is widely known — not hidden.
3. These sources can be used for verification after a company is found through a hidden source, but not for discovery.
4. If the only evidence for a candidate is one of these sources, note it explicitly in the `source_url` and `why_non_obvious` fields, and expect a downgrade from the anti-mainstream filter.
